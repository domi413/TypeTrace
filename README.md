# TypeTrace

This is the GitHub repository of the TypeTrace sourcecode

## Backend

- Pep8 standard

- Run: `sudo python -m typetrace`
- Debug: `sudo python -m typetrace --debug`

- Format: `ruff check file.py --select ALL --ignore "T201"`
  - `T201` ignores print statements
- Type checking: Use `mypy` to check if the type checking is correct
